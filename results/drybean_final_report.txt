【Part 1】数据分析 (5%)
────────────────────────────────────────────────────────────────

数据集: Dry Bean Dataset
来源: UCI Machine Learning Repository
描述: 7种干豆的16个形态特征（Area, Perimeter, MajorAxisLength等）
样本: 训练集9527 + 验证集1347 + 测试集2737 = 13611

数据污染发现:
  1. 类别标签: 25种乱码→7种标准类（大小写、拼写错误D3RMAS0N、尾部空格）
  2. 特征污染: Solidify/Compactness含非数值字符串
  3. 缺失值: 约0.5%
  4. 异常值: Area等特征存在离群点
  5. 特征高相关: 16特征中6对r>0.95


【Part 2】数据处理 (30%)
────────────────────────────────────────────────────────────────

处理流程:
  1. 类别清洗: str.strip() + 映射字典统一7类
  2. 字符串转数值: pd.to_numeric(errors='coerce')
  3. 缺失值填充: 中位数
  4. 特征缩放: StandardScaler
  5. 特征选择: 16→10个（去除Perimeter, ConvexArea等高相关特征）
  6. 保存: train_clean.csv, val_clean.csv, test_clean.csv
  7. 保存Scaler: models/drybean_scaler.pkl


【Part 3】多算法实验对比 (30%)
────────────────────────────────────────────────────────────────

采用4种算法（含XGBoost为课上未讲过的）:

算法                    训练acc   验证acc   测试acc   训练时间   推理时间   过拟合
─────────────────────────────────────────────────────────────────────────────
Logistic Regression    92.51%    92.35%    91.85%    0.17s     6.0ms     0.65%
SVM (RBF)              92.99%    92.95%    92.88%    5.24s     4526ms    0.11%
KNN (k=5)              93.78%    91.69%    91.74%    0.04s     785ms     2.03%
XGBoost ⭐             99.58%    92.65%    92.58%    4.31s     116ms     7.00%

鲁棒性测试（添加高斯噪声）:
  σ=0.0:  LR=91.9%  SVM=92.9%  KNN=91.7%  XGB=92.6%
  σ=0.2:  LR=89.3%  SVM=91.0%  KNN=90.7%  XGB=88.8%
  σ=0.5:  LR=79.4%  SVM=85.5%  KNN=84.8%  XGB=75.2%
  σ=1.0:  LR=60.2%  SVM=68.2%  KNN=70.2%  XGB=54.6%

结论:
  - 准确率: SVM最高(92.88%)，XGBoost接近(92.58%)
  - 速度: LR最快(6ms)，SVM最慢(4.5s)
  - 鲁棒性: SVM和KNN抗噪声最强，XGBoost最弱
  - 过拟合: SVM最小(0.11%)，XGBoost最大(7%)
  - ⭐ XGBoost为课堂上未讲过的算法（加分项）

【Part 5】课程总结 (5%)
────────────────────────────────────────────────────────────────

通过本课程的学习，主要收获如下:

1. 掌握了完整的机器学习项目流程：
   从数据分析→数据清洗→特征工程→模型训练→结果分析的全流程实践能力。

2. 理解了不同算法的特性差异：
   - 逻辑回归: 简单快速，适合线性可分问题
   - SVM: 核技巧强大，但推理速度随支持向量数增加变慢
   - KNN: 无需训练，但推理需全部样本计算
   - XGBoost: 梯度提升能力强，但易过拟合

3. 数据处理的重要性：
   原始数据存在标签污染、缺失值、异常值等问题，必须经过清洗才能训练。
   "脏数据进，脏数据出"——数据处理占整个项目60%以上的工作量。

4. 系统工程化能力：
   学会了统一命令行接口、模块化设计、模型保存与加载、结果可视化等工程实践。

5. 建议：
   - 增加深度学习相关实验（CNN、RNN等）
   - 引入更多真实世界的高维数据集
   - 增加模型部署和API服务相关教学
